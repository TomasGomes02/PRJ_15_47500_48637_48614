Detailed Description

Class ApiExtension.

This class extends the TicketApiController class to use our own endpoints.

◆ requestApiKey()

requestApiKey ( $format)

Function executed when the endpoint/url requestApiKey/tickets is called.

Verifies if the staff specified exists, if the person making the request is an admin, and if the password inserted matches the password of the person making the request.

Adds a new API key and deactivates the old one to the specified staff, if no staff is specified, the API key is added to the admin making the request.

Makes a response with the new API key. If there are errors response has code 500 and specified error.

Parameters
object$formatJson sent in the HTTP body.
